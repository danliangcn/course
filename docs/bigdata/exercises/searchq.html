数据检索与选题探索作业

Published

September 15, 2025

1 作业目的

熟悉公共管理领域常见的大数据来源，掌握数据获取与理解的基本方法，并通过结合相关文献，初步提出潜在的研究问题，为课程论文选题和设计奠定基础。

2 作业要求

  • 文献阅读：检索公共管理、公共政策、城市管理等相关领域采用大数据分析的相关文献，选择至少2篇（中英文均可）进行阅读，简要总结其研究问题、数据来源、研究方法和主要发现。

  • 数据源探索：从开放数据清单或其他数据来源中选择至少1个数据源，尝试获取并浏览数据，说明其内容、变量特征和可能的研究价值。

  • 研究问题提出：结合文献和数据探索，提出 2-3个潜在的研究问题，并简要说明：为什么这个问题在公共管理中有价值？哪类数据可能支持回答？以及可能的分析方法。

  • 提交形式：Word 或 PDF 文档（1500字左右），参考文献引用规范，10月26日中午12点前发到助教邮箱

3 评分标准

  • 文献总结（30%）：准确概括研究内容，突出数据与方法。

  • 数据源探索（30%）：数据介绍清晰，能说明其潜在研究价值。

  • 研究问题提出（30%）：问题具体、有研究意义，体现数据驱动思考。

  • 结构与表达（10%）：逻辑清晰，引用格式规范。

4 开放数据源参考

4.1 国内开放数据源

1. 政府与公共服务

  • 中国政府网“数据”栏目 http://data.gov.cn/

    • 全国性政府数据平台，涵盖经济、社会、民生、交通等多领域。

  • 国家统计局数据 http://data.stats.gov.cn/

    • 各类宏观统计指标（人口、经济、教育、医疗、环境）。

  • 地方政府开放数据平台

    • 如：北京开放数据平台（https://data.beijing.gov.cn/），上海开放数据服务网（https://data.sh.gov.cn/）
    • 可用于城市治理、公共服务供给公平研究。

2. 环境与可持续发展

  • 中国环境监测总站数据 http://www.cnemc.cn/

    • 空气质量、水质、噪音监测数据。

  • 中国碳排放数据库（CEADs） http://www.ceads.net.cn/

    • 各省、市碳排放清单数据。

  • 国家地球系统科学数据中心 https://www.geodata.cn

    • 多时空尺度综合性地球系统科学数据库群。

3. 社会与民生

  • 中国知网-中国经济社会大数据研究平台 https://data.cnki.net//

    • 统计、普查、调查资料等。

  • 中国健康与养老追踪调查（CHARLS） http://charls.pku.edu.cn/

    • 大规模面板数据，关注健康、养老、社会保障。

  • 中国家庭追踪调查（CFPS） http://www.isss.pku.edu.cn/cfps/

    • 涉及教育、就业、家庭结构、收入分配。

4. 风险管理与应急

  • 中国地震局科学数据中心 http://data.earthquake.cn/

    • 地震监测数据。

  • 中国气象科学数据共享服务网 http://data.cma.cn/

    • 气象监测、灾害预警数据。

  • 微博数据（需通过API或爬取）

    • 可用于应急舆情、社会风险感知研究。

4.2 国际开放数据源

1. 政府与公共政策

  • World Bank Open Data https://data.worldbank.org/

    • 全球发展指标，适合跨国比较。

  • OECD Data https://data.oecd.org/

    • 涵盖教育、劳动力、环境、治理等主题。

  • UN Data https://data.un.org/

    • 联合国各类统计数据库。

2. 环境与可持续发展

  • NASA Earthdata https://earthdata.nasa.gov/

    • 遥感与气候数据。

  • European Environment Agency (EEA) Data https://www.eea.europa.eu/data-and-maps

    • 欧洲环境、污染、能源数据。

  • Global Carbon Atlas http://www.globalcarbonatlas.org/

    • 全球碳排放数据。

3. 社会与民生

  • Pew Research Center Datasets https://www.pewresearch.org/download-datasets/

    • 公共舆论、社会价值观、治理调查。

  • European Social Survey (ESS) https://ess-search.nsd.no/

    • 欧洲各国社会态度与政策数据。

  • IPUMS (Integrated Public Use Microdata Series) https://ipums.org/

    • 全球人口普查与调查微观数据。

4. 风险管理与应急

  • EM-DAT: The International Disaster Database https://www.emdat.be/

    • 全球自然灾害数据库。

  • CrisisNET (Archive) https://crisis.net/

    • 人道危机、冲突事件数据。

  • Twitter API https://developer.x.com/en/docs/x-api

    • 可用于应急舆情分析。
